repositorio sobre programas que utilizam api's

api_interna:

é uma api web para um programa interno utilizando web que consulta o nome da CID pelo codigo da CID e pelo idioma de entrada
Funciona internamente, repare que é localhost, você terá de instalar seu banco de dados e conectar a ele
Foi utilizado o MariaDB como banco de dados e HeidiSQL para visualização

Inicie primeiro o server.js "node server.js" no terminal
Em seguida o diretório em si, funciona mesmo em diferentes portas

Atenção:
Há um SQL basico de exemplo para voce colocar os dados e ir manejando
